## Déployer sur Vercel

### Option 1 — Via GitHub (recommandé)
1. Crée un repo GitHub `telopex`
2. Push ce dossier
3. Va sur vercel.com → Import Git Repository
4. Sélectionne le repo → Deploy

### Option 2 — Via Vercel CLI (Termux)
```bash
npm install -g vercel
vercel login
vercel deploy
```

### Option 3 — Drag & Drop
1. Va sur vercel.com/new
2. Glisse le dossier complet
3. Deploy
